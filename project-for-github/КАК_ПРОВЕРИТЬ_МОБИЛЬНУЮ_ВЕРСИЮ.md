# Как проверить мобильную версию сайта локально

## Способ 1: DevTools браузера (Рекомендуется)

Это самый удобный способ проверить мобильную версию без реального телефона.

### Шаги:

1. **Запустите Flask приложение:**
   ```bash
   cd newyear_book_rewrite
   python app.py
   ```
   Приложение будет доступно по адресу: `http://localhost:5000`

2. **Откройте браузер Chrome или Firefox:**
   - Откройте `http://localhost:5000`

3. **Откройте DevTools (Инструменты разработчика):**
   - **Chrome/Edge:** Нажмите `F12` или `Ctrl+Shift+I` (Windows/Linux) / `Cmd+Option+I` (Mac)
   - **Firefox:** Нажмите `F12` или `Ctrl+Shift+I` (Windows/Linux) / `Cmd+Option+I` (Mac)

4. **Включите режим адаптивного дизайна:**
   - **Chrome/Edge:** Нажмите `Ctrl+Shift+M` (Windows/Linux) / `Cmd+Shift+M` (Mac)
   - Или нажмите на иконку телефона/планшета в панели DevTools
   - **Firefox:** Нажмите `Ctrl+Shift+M` или нажмите на иконку телефона в панели

5. **Выберите устройство:**
   - В верхней панели выберите устройство из списка (например, iPhone 12 Pro, iPhone SE, Samsung Galaxy S20 и т.д.)
   - Или установите свой размер: например, 375x667 (iPhone SE) или 390x844 (iPhone 12 Pro)

6. **Обновите страницу:**
   - Нажмите `F5` или `Ctrl+R` чтобы применить изменения

### Полезные настройки в DevTools:

- **Throttling (Ограничение скорости):** Можно выбрать "Fast 3G" или "Slow 3G" для проверки на медленном интернете
- **Device Pixel Ratio:** Убедитесь, что установлен правильный коэффициент (обычно 2 или 3 для современных телефонов)
- **Touch Emulation:** Включите эмуляцию сенсорного экрана

## Способ 2: Изменение размера окна браузера

1. Запустите приложение как описано выше
2. Откройте `http://localhost:5000`
3. Уменьшите ширину окна браузера до 480px или 768px
4. Медиа-запросы автоматически применят мобильные стили

## Способ 3: Доступ с реального мобильного устройства в локальной сети

Если вы хотите проверить на реальном телефоне:

1. **Узнайте локальный IP адрес вашего компьютера:**
   - **Windows:** Откройте командную строку и выполните `ipconfig`
     - Найдите "IPv4 Address" (например, 192.168.1.100)
   - **Mac/Linux:** Выполните `ifconfig` или `ip addr`
     - Найдите IP адрес (обычно начинается с 192.168.x.x или 10.x.x.x)

2. **Убедитесь, что Flask приложение доступно из сети:**
   - В `app.py` уже установлено `host='0.0.0.0'`, что означает доступ из сети
   - Если нет, измените на `app.run(debug=True, host='0.0.0.0', port=5000)`

3. **Убедитесь, что устройства в одной Wi-Fi сети:**
   - Компьютер и телефон должны быть подключены к одной сети Wi-Fi

4. **Откройте на телефоне:**
   - В браузере телефона введите: `http://ВАШ_IP:5000`
   - Например: `http://192.168.1.100:5000`

5. **Возможные проблемы:**
   - Если не открывается, проверьте настройки файрвола
   - На Windows может потребоваться разрешить доступ для Python в файрволе

## Быстрая проверка размеров экранов

Для проверки конкретных размеров в DevTools:

- **iPhone SE (маленький телефон):** 375 x 667
- **iPhone 12/13/14:** 390 x 844
- **iPhone 12/13/14 Pro Max:** 428 x 926
- **Samsung Galaxy S20/S21:** 360 x 800
- **iPad:** 768 x 1024

Эти размеры можно ввести вручную в режиме адаптивного дизайна.
