# Как загрузить всю папку на GitHub заново

## ⚠️ ВАЖНО: Не нужно удалять репозиторий!

Просто обновите файлы. Есть несколько способов:

───────────────────────────────────────────────────────────
  СПОСОБ 1: GitHub Desktop (САМЫЙ ПРОСТОЙ)
───────────────────────────────────────────────────────────

1. **Скачайте GitHub Desktop** (если еще не установлен):
   https://desktop.github.com/

2. **Откройте GitHub Desktop**
   - File → Add Local Repository
   - Выберите папку `project-for-github`
   - Или File → Clone Repository → выберите ваш репозиторий

3. **Все изменения появятся автоматически**
   - Напишите комментарий: "Обновление мобильной версии"
   - Нажмите "Commit to main"
   - Нажмите "Push origin"

Готово! Все файлы загружены одной кнопкой.

───────────────────────────────────────────────────────────
  СПОСОБ 2: Через Git командную строку
───────────────────────────────────────────────────────────

1. **Откройте командную строку** в папке `project-for-github`

2. **Если это первый раз:**
   ```bash
   git init
   git remote add origin https://github.com/Viktor5460/NewYearPlan.git
   ```

3. **Добавьте все файлы:**
   ```bash
   git add .
   git commit -m "Обновление мобильной версии"
   git push -u origin main
   ```

4. **Если репозиторий уже существует:**
   ```bash
   git add .
   git commit -m "Обновление мобильной версии"
   git push
   ```

───────────────────────────────────────────────────────────
  СПОСОБ 3: Через веб-интерфейс (загрузка ZIP)
───────────────────────────────────────────────────────────

1. **Зайдите на GitHub:**
   https://github.com/Viktor5460/NewYearPlan

2. **Нажмите "Add file" → "Upload files"**

3. **Перетащите все файлы из папки `project-for-github`**

4. **Нажмите "Commit changes"**

───────────────────────────────────────────────────────────
  ⚠️ НЕ ДЕЛАЙТЕ:
───────────────────────────────────────────────────────────

❌ НЕ удаляйте репозиторий полностью
❌ НЕ создавайте новый репозиторий

Просто обновите файлы - это безопаснее!
