# Обновление мобильной версии на GitHub

## Что было изменено:

1. **templates/mailbox.html** - Обновлен viewport для правильного масштабирования
2. **templates/letters.html** - Обновлен viewport для правильного масштабирования
3. **static/css/letters.css** - Добавлены мобильные стили:
   - Уменьшен масштаб всех элементов на мобильных устройствах
   - Дата теперь под временем (не справа) на мобильных
   - Текст на письмах опущен ниже
   - Левая колонка в режиме осмотра уменьшена в 1.5 раза на мобильных
4. **static/js/letters.js** - Обновлена логика ширины колонок для мобильных устройств

## Инструкция по загрузке на GitHub:

### Способ 1: Через веб-интерфейс GitHub (самый простой)

1. Откройте ваш репозиторий на GitHub: https://github.com/Viktor5460/NewYearPlan (или ваш репозиторий)

2. Для каждого измененного файла:
   - Нажмите на файл (например, `templates/mailbox.html`)
   - Нажмите иконку карандаша (Edit) справа вверху
   - Скопируйте содержимое файла из папки `project-for-github` и вставьте в редактор
   - Прокрутите вниз и нажмите "Commit changes"
   - Напишите комментарий: "Обновление мобильной версии"

3. Повторите для всех 4 файлов:
   - `templates/mailbox.html`
   - `templates/letters.html`
   - `static/css/letters.css`
   - `static/js/letters.js`

### Способ 2: Через Git командную строку (если установлен Git)

1. Откройте командную строку в папке `project-for-github`

2. Выполните команды:
   ```bash
   git add templates/mailbox.html templates/letters.html static/css/letters.css static/js/letters.js
   git commit -m "Обновление мобильной версии: адаптивный дизайн, уменьшенный масштаб"
   git push
   ```

### Способ 3: Загрузка через GitHub Desktop (если установлен)

1. Откройте GitHub Desktop
2. Выберите ваш репозиторий
3. Файлы должны автоматически появиться в списке изменений
4. Напишите комментарий: "Обновление мобильной версии"
5. Нажмите "Commit to main" и "Push origin"

## После загрузки на GitHub:

1. **Render автоматически перезапустит деплой** (если настроен автодеплой)
2. Или **перезапустите вручную**:
   - Зайдите на https://dashboard.render.com
   - Найдите ваш сервис
   - Нажмите "Manual Deploy" → "Deploy latest commit"

## Проверка:

После деплоя проверьте сайт на мобильном устройстве или в режиме эмуляции мобильного:
- Откройте сайт на телефоне
- Или используйте DevTools браузера (F12 → Ctrl+Shift+M)
- Проверьте, что элементы не накладываются друг на друга
- Проверьте, что дата под временем на письмах
- Проверьте режим осмотра - левая колонка должна быть уже
