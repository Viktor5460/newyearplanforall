═══════════════════════════════════════════════════════
  📋 КОМАНДЫ ДЛЯ GITHUB:
═══════════════════════════════════════════════════════

1. Создайте репозиторий на GitHub.com

2. В этой папке выполните:
   
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/ВАШ_НИКНЕЙМ/newyear-plans.git
   git push -u origin main

3. После загрузки в GitHub перейдите к папке:
   render-deploy-instructions
   
   Там инструкции как подключить GitHub к Render

═══════════════════════════════════════════════════════
  ⚠️ ВАЖНО:
═══════════════════════════════════════════════════════

Render НЕ требует загрузки файлов!
Он автоматически берет код из GitHub репозитория.
