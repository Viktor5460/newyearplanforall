# Как перезапустить Flask приложение локально

## Быстрый способ:

1. **Остановите текущий процесс Flask:**
   - Нажмите `Ctrl+C` в терминале, где запущен Flask

2. **Запустите заново:**
   ```bash
   cd newyear_book_rewrite
   python app.py
   ```

3. **Откройте в браузере:**
   ```
   http://localhost:5000
   ```

## Если изменения CSS не видны (кэш браузера):

### Способ 1: Жесткое обновление страницы
- **Windows/Linux:** `Ctrl + Shift + R` или `Ctrl + F5`
- **Mac:** `Cmd + Shift + R`

### Способ 2: Очистить кэш браузера
- **Chrome/Edge:** 
  - Нажмите `F12` (DevTools)
  - Правой кнопкой на кнопку обновления
  - Выберите "Очистить кэш и жесткая перезагрузка"

- **Firefox:**
  - Нажмите `Ctrl + Shift + Delete`
  - Выберите "Кэш"
  - Нажмите "Очистить сейчас"

### Способ 3: Открыть в режиме инкогнито
- Откройте новое окно в режиме инкогнито/приватном
- Перейдите на `http://localhost:5000`

## Проверка:

После перезапуска и обновления страницы изменения должны быть видны!
