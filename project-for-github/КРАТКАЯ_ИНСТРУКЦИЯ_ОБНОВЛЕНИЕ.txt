───────────────────────────────────────────────────────────
  СПОСОБ 1: Через сайт GitHub (САМЫЙ ПРОСТОЙ)
───────────────────────────────────────────────────────────

1. Откройте: https://github.com/Viktor5460/NewYearPlan

2. Для каждого файла:
   - Нажмите на файл
   - Нажмите иконку ✏️ (Edit) справа вверху
   - Откройте файл из папки project-for-github на вашем компьютере
   - Скопируйте ВСЁ содержимое (Ctrl+A, Ctrl+C)
   - Вставьте в редактор GitHub (Ctrl+V)
   - Внизу напишите: "Обновление мобильной версии"
   - Нажмите "Commit changes"

3. Повторите для всех 4 файлов!
